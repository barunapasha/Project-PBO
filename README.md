# Project-PBO
Repository ini bertujuan untuk membagikan file project PBO dan bertujuan untuk membuat project ini lebih tersusun

**UPartner: Platform Kolaborasi Mahasiswa dan Dosen**
UPartner adalah platform web yang memfasilitasi pembuatan, pengelolaan, dan partisipasi dalam proyek kolaboratif antara mahasiswa dan dosen. Platform ini dirancang untuk membantu mahasiswa dan dosen bekerja sama secara efektif dalam berbagai proyek, seperti penelitian, pengembangan, dan pengabdian kepada masyarakat.

Fitur:
- Pendaftaran dan Autentikasi: Pengguna dapat mendaftar dan masuk ke platform menggunakan email dan password.
- Pencarian Proyek: Mahasiswa dapat mencari lowongan proyek berdasarkan jurusan di Universitas Pertamina.
- Pendaftaran Proyek: Mahasiswa dapat mendaftar ke proyek yang mereka minati dan memberikan alasan pendaftaran.
- Manajemen Proyek: Dosen dapat membuat proyek, menerima mahasiswa, dan mengelola proyek mereka.
- Komunikasi: Mahasiswa dan dosen dapat berkomunikasi melalui fitur chat untuk mendiskusikan proyek mereka.
- Informasi Profil: Pengguna dapat melihat informasi profil mereka, termasuk nama, NIM/NIP, jurusan, email, proyek yang telah diselesaikan, dan status proyek (sedang dalam proyek, sedang tidak dalam proyek).
- Keamanan: Sistem menyimpan data pengguna secara ketat dan memastikan data pengguna tidak hilang dan bocor.
- Portabilitas: Sistem responsif ketika pengguna menggunakan perangkat yang berbeda.
- Integritas Data: Sistem selalu diperbarui terkait perubahan pada proyek, seperti saat anggota proyek terpenuhi, proyek dihapus, atau proyek dibatalkan.

Persyaratan Sistem:
- Browser web modern (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, dll.)
- Koneksi internet yang stabil
